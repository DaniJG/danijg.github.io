An extension for the [sinon.js](https://sinonjs.org/) mocking library. It makes easier to write unit tests for code that uses the MongoDB [Node.js driver](https://mongodb.github.io/node-mongodb-native/), allowing you to create stubs and spys for the client, db and collection objects.

Install with `npm i -D sinon-mogo`. Read more [in KaizenDorks](https://kaizendorks.github.io/2019/01/30/sinon-mongo/).
